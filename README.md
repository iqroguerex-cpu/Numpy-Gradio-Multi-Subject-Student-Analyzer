## 🚀 Overview

An interactive **Student Performance Dashboard** built with **Streamlit**, **NumPy**, and **Matplotlib**.

This app analyzes multi-subject scores and visualizes insights using charts, making it a strong **data analytics portfolio project**.

---

## ✨ Features

* 🎲 Generate student scores (3 subjects)
* 📊 Student total & average analysis
* 📈 Subject-wise performance insights
* 🏆 Student ranking system
* 🔍 Filter students by average
* 📊 **Interactive charts (bar graphs)**

---

## 🛠️ Tech Stack

* Python 3.x
* Streamlit
